## How To Use This Catalog

When the user asks for references:

1. Pick the most relevant category first
2. Name 3 to 5 concrete sources
3. Explain what each source is good for
4. Extract the pattern, trend, or structure worth borrowing
5. Explain what not to copy blindly

When the user asks for a concept:

1. Pull references from at least one visual category
2. Pull UX evidence from `NNGroup`, `Baymard Institute`, or another strong research source when relevant
3. Map the result to a design system or component library if implementation matters
